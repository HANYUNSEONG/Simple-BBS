# Simple BBS

공부하며 간단한 게시판을 만들었습니다.

## 공통

- Typescript

## Client

- NextJS
- Emotion
- Recoil
- React Query
- Toast UI Editor

## Server

- NestJS
- TypeORM
- PostgreSQL

## 기능

### 계정

- 로그인/로그아웃
- 회원가입

### 게시판

- 게시글 쓰기
  - 이미지 업로드
  - Markdown Editor 사용
  - 공개/비공개 선택
- 게시글 읽기
  - Toast UI Editor에서 제공하는 Viewer 사용
- 게시글 리스트
  - 무한 스크롤링 구현
- 게시글 삭제
